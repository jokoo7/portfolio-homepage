## Code

### Inline Code

Use the `useState` hook to manage state in React components.

### Code Block

```javascript
function HelloWorld() {
  const [count, setCount] = useState(0);

  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={() => setCount(count + 1)}>Increment</button>
    </div>
  );
}
```

## Interactive Elements

<details>
  <summary>Click to expand - What is MDX?</summary>

MDX is a format that lets you seamlessly write JSX in your Markdown documents.
You can import components, such as interactive charts or alerts, and embed
them within your content. **Benefits:** - Write interactive content - Use
React components - Keep markdown simplicity

</details>
