## Tips Aksesibilitas

- Selalu gunakan `alt` pada gambar.
- Hindari `target="_blank"` tanpa `rel="noopener noreferrer"`.
- Gunakan heading (`#`, `##`, `###`, dst.) secara hierarkis.

## Catatan Penutup

File MDX ini:

- Tidak memakai tag HTML manual seperti `<p>` atau `<table>`.
- Sepenuhnya kompatibel dengan Markdown parser MDX.
- Bisa dikembangkan dengan komponen React jika diinginkan.
